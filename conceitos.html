O que é um software?

É um conjunto de instruções (programas) que dizem ao computador o que fazer.

O que é um sistema operacional?

É o software que gerencia o hardware do computador e permite a execução de outros programas (ex: Windows, Linux).

O que é uma licença de software?

É um contrato que define como o software pode ser usado, copiado ou distribuído.

O que é um Software Licenciado?

É um software que só pode ser usado conforme os termos da licença, muitas vezes exigindo pagamento ou registro.

O que é um Software livre?

É um software que pode ser usado, modificado e distribuído livremente por qualquer pessoa.

O que é um Software Proprietário?

É um software que pertence a uma empresa ou pessoa, com uso restrito por licença.

O que é um software de Sistema?

É um software que gerencia os recursos do computador, como sistemas operacionais e drivers.

O que é um software Aplicativo?

É um programa usado para realizar tarefas específicas, como editores de texto, navegadores, etc.

O que é um software de Programação?

São ferramentas usadas para criar outros softwares, como editores de código, compiladores e IDEs.

O que é um software de diagnóstico de software ?

É um programa que detecta problemas ou erros em outros softwares.

O que é um software de diagnóstico de hardware ?

É um programa que testa e verifica se os componentes físicos do computador estão funcionando corretamente.
