1.1 Componentes internos

Os componentes internos de um computador são as peças que trabalham em conjunto para que o computador funcione, como a placa-mãe, o processador, a memória RAM, os dispositivos de armazenamento (HD, SSD) e a fonte de alimentação.

1.1.4 Fonte de alimentação

Uma fonte de alimentação é um dispositivo elétrico que converte a energia elétrica de uma fonte (como a rede elétrica) para uma tensão e corrente adequadas para alimentar um dispositivo ou sistema. Em outras palavras, ela fornece a energia elétrica necessária para o funcionamento de um circuito ou equipamento.

1.3 Interfaces de entrada e saída

As interfaces de entrada e saída (E/S) são componentes cruciais em sistemas computacionais, facilitando a comunicação entre a CPU e os dispositivos periféricos.

1.3.1 Interfaces sem fio

Interfaces sem fio, também conhecidas como interfaces de rede sem fio (Wireless Network Interface Controllers - WNIC), são dispositivos que permitem a comunicação entre dispositivos eletrônicos sem a necessidade de cabos.

1.3.3 Rede (Ethernet)

Uma rede Ethernet é uma tecnologia de rede de computadores com fio amplamente utilizada para conectar dispositivos em redes locais (LAN), redes metropolitanas (MAN) e redes de longa distância (WAN).

1.3.4 VGA

VGA é a sigla para "Video Graphics Array", um padrão de exibição de vídeo e tipo de conexão amplamente utilizado em computadores. Introduzido pela IBM em 1987, o VGA tornou-se o padrão gráfico padrão para PCs e lançou as bases para os monitores de computador modernos.

1.3.5 HDMI

HDMI, que significa High-Definition Multimedia Interface (Interface Multimídia de Alta Definição), é uma interface de áudio/vídeo digital que permite a transmissão de dados de vídeo e áudio de um dispositivo para outro. É um padrão universal para conectar dispositivos como Smart TVs, computadores, consoles de jogos, Blu-rays e outros equipamentos

1.3.6 USB

USB (Universal Serial Bus) é um padrão de conexão por cabo que permite transmissão de dados entre dispositivos variados, como celulares, PCs, TVs e acessórios.
